iOS 线程同步: @synchronized指令

Jun 2nd, 2015

1.@synchronized指令是Objective-C代码中快捷创建互斥锁的一种方式；

2.@synchronized指令像任何其他的互斥锁一样，防止同一时间不同的线程获取相同的锁。

@synchronized指令的使用很简单，只需要传一个参数即可：

@synchronized(key) { 
    // thread-safe code goes here
